¿Por qué dos programas?

Ambos programas están diseñados para complementarse, asegurando tanto la coherencia curricular nacional como la pertinencia educativa local.

El Programa Sintético proporciona la base común para todo el país, mientras que el Programa Analítico permite la autonomía docente para ajustar el currículo a la realidad local.

Definición y Finalidad

Aspecto Programa Sintético Programa Analítico
Definición Es el primer nivel de concreción curricular que establece los elementos centrales del plan de estudios. Es el segundo nivel de concreción curricular, desarrollado por los colectivos docentes según el contexto escolar.
Finalidad Proporcionar una base común nacional para la educación básica. Adaptar los contenidos a las necesidades específicas de cada comunidad escolar.
🇲🇽

Nacional

El Programa Sintético garantiza estándares comunes en todo México

🏘️

Local

El Programa Analítico responde a la diversidad de cada comunidad

Flexibilidad y Elaboración

Aspecto Programa Sintético Programa Analítico
Flexibilidad Proporciona lineamientos generales, pero permite contextualización. Totalmente flexible y contextualizado a la realidad de cada escuela y comunidad.
Elaboración Diseñado por la Secretaría de Educación Pública (SEP). Diseñado por maestros y colectivos docentes en cada escuela.

¿Quién lo crea?

Programa Sintético: Expertos de la SEP a nivel federal

Programa Analítico: Los docentes de cada escuela en trabajo colaborativo

Estructura y Participación Docente

Aspecto Programa Sintético Programa Analítico
Estructura Se organiza en Campos Formativos, Contenidos y Procesos de Desarrollo de Aprendizaje. Se construye a partir del análisis del contexto, la codiseñación y la planificación didáctica.
Participación Docente Ofrece una guía general, permitiendo la autonomía del docente para su aplicación. Exige una participación activa del colectivo docente en su elaboración.

Nivel de Concreción y Perfil de Egreso

Aspecto Programa Sintético Programa Analítico
Nivel de Concreción Es un documento base con contenido nacional. Es una propuesta específica ajustada a la realidad de cada escuela y comunidad.
Relación con el Perfil de Egreso Asegura que los contenidos nacionales mantengan coherencia con el perfil de egreso. Adapta los aprendizajes esperados al contexto de cada estudiante y comunidad.

Del General al Específico

1° Nivel: Programa Sintético → Elementos comunes nacionales

2° Nivel: Programa Analítico → Adaptación contextualizada

3° Nivel: Planificación de aula → Implementación diaria

Evaluación y Ejemplo de Aplicación

Aspecto Programa Sintético Programa Analítico
Vinculación con la Evaluación Presenta lineamientos generales para la evaluación formativa. Permite definir estrategias de evaluación formativa adaptadas a la realidad del aula.
Ejemplo de Aplicación Propone que, en Lenguaje, los estudiantes desarrollen habilidades de comunicación oral y escrita. Los docentes definen actividades específicas, como la narración de historias locales o la lectura de textos comunitarios.
